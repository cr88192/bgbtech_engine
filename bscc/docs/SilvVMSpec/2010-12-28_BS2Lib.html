Aliased Types
A base type may be aliased to a class or struct, which means that this class will be used to handle operations on said type, and said type may behave as if it were an instance of said class.

Aliasing will work by creating a new instance of the class or struct and calling the constructor with the aliased value, followed by invoking whatever method is needed, and if applicable, deleting the object.
