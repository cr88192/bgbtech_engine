BGB Tech Engine Overview



Major Libraries




BTGECM: BGBTech Common


Deals with structure shared between the client and server ends.

BTGESV: BGBTech Server


Deals mostly with the server end, and with things like game logic.


PDGL: Lower-Level Rendering Infrastructure


Deals mostly with things related to getting video and sound to work, loading textures, ...


LBXGL: Higher-Level Rendering Infrastructure


Deals mostly with higher-level rendering tasks, such as rendering the world and various objects.
